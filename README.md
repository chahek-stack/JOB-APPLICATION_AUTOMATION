# JOB-APPLICATION_AUTOMATION
This project automates the process of tracking job application by integrating notion,googlesheet and gmail using MAKE.Including real time data sync, email alert, HTTP BASED JSON data transfer to a webhook,simulating real world API integration workflows.
* JOB APPLICATION AUTOMATION SYSTEM 
* OVERVIEW
This project automates job application tracking using notion , google sheets , gmail, and MAKE.COM
* FEATURES
  - Automated job tracking system
  - Real-time data sync( notion TO google sheets)
  - Email notifications via Gmail
  - HTTP  API integration using JSON
  * TOOLS USED
    - Notion
    - Google sheets
    - Gmail
    - MAKE.COM
    - Webhook.site
  * WORKFLOW
    - ADD JOB APPLICATION IN NOTION
    - Data syncs to google sheets
    - Email notification is triggered
    - Data is sent via HTTP request (JSON) to webhook
